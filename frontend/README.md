:: Overview
This project is a full-stack web application designed for a pet grooming business. Customers can create accounts, add pets, and book grooming services, while admins can manage bookings, services, and customer data. Designed to deliver a clean, user-friendly experience for both customers and administrators.

:: Features

1. Customer Portal
   - Sign up / log in securely
   - Add pets and book grooming services
   - View booking history and status
2. Admin Dashboard
   - Manage customer bookings (approve, cancel, delete)
   - Add or update services
   - View analytics and summaries
3. Core Features
   - Responsive design for desktop and mobile
   - Secure authentication with JWT
   - PostgreSQL database with relational schema
   - RESTful API built with Express.js

:: Tech Stack
Frontend

- React + Vite
- TailwindCSS v4
  Backend
- Node.js + Express
- PostgreSQL (via pg)
- JWT + bcrypt for authentication
- dotenv for environment configuration
- cors for frontend-backend communication
